# DeadEnd

Remake of DeadEnd II, a puzzle game initially developed by Wolfgang Thaller (released for the Macintosh in 1998)

## Installation

Clone the project:

```
$ git clone https://github.com/arfeo/DeadEnd.git && cd DeadEnd
```

Run:

```
$ yarn
```

```
$ gulp
```

Build:

```
$ gulp build
```
